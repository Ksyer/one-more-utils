# one-more-util

**说明：**

开发中的一些自定义通用函数库。

## 功能描述

**函数相关：**

- call()& apply()& bind()

**数组相关：**

- chunk

## 用法

```bash
# npm 安装方式
npm i one-more-util

# yarn 安装方式
yarn add one-more-util

```

JS 引入方式

```html
<script src="../../dist/one-more-util.min.js"></script>
<script>
  console.log(omUtils) // 直接 omUtils. 使用
</script>
```
